Neon PostgreSQL
Bepul serverless PostgreSQL — Neon.tech
1. Project yaratish
  1. neon.tech → ga kiring
  2. Sign Up → GitHub bilan kirish (tez)
  3. Create Project bosing
  4. Project name: devprep
  5. Region: AWS US East (N. Virginia) yoki yaqinroq
  6. PostgreSQL version: 16
  7. Create Project bosing
2. Connection string olish

Project yaratilgandan keyin avtomatik ko'rinadi. Agar ko'rinmasa: Dashboard → Connection Details

Connection string 2 xil bo'ladi — ikkalasi ham kerak!
📌 Pooled Connection (DATABASE_URL):
server/.envenv
# Connection pooler orqali (ko'p connection uchun)
DATABASE_URL="postgresql://USER:PASS@ep-xxx.us-east-2.aws.neon.tech/neondb?sslmode=require&pgbouncer=true"
📌 Direct Connection (DIRECT_URL):
server/.envenv
DIRECT_URL="postgresql://USER:PASS@ep-xxx.us-east-2.aws.neon.tech/neondb?sslmode=require"
⚠️ Bu connection string faqat Neon dashboard'da ko'rinadi. Hoziroq nusxalab saqlang!
